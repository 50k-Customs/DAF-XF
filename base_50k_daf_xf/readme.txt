DAF XF 105 modified by 50keda featuring:

- 12+ new interior variants
- 4 spoke steering wheel
- Complete UK variant
- Rigid chassis (compatible with Flemmings BDF Tandem Mod):
  - 6x2
  - 4x2
- New cabins without spoilers
- 8 types of rigid chassis equipments including new sideskirts and Thermo King UTS
- Extra upgrades: roof spoiler, roof grill, front grill, bottom grill, front fenders and more

### CHANGELOG FOR LATEST VERSION

- Fixed 4x2 chassis bug with marker lights
- Added 3 new accessories
- Updated DAF Tuning Pack compatibility package: tables are available in new interiors

### WARNING FOR PRE v4.0 USERS

If your previous version of this mod is lower than v4.0 then make sure
to sell all of your 50k DAF XF 105 trucks before using this version!
